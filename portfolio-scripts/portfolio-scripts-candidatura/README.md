# Portefólio de Scripts para  Técnico de Sistemas e Tecnologias de Informação

Este repositório reúne scripts simples e úteis, pensados para demonstrar competências práticas alinhadas com funções de suporte técnico, manutenção de sistemas, troubleshooting, inventário e operações básicas de administração.

## Objetivo
Mostrar capacidade de:
- automatizar tarefas repetitivas;
- recolher informação técnica de equipamentos;
- executar verificações básicas de conectividade;
- apoiar tarefas de backup e suporte ao utilizador;
- trabalhar com Windows e Linux.

## Estrutura
- `powershell/`
  - `windows_system_inventory.ps1` — recolha de inventário básico do sistema Windows;
  - `backup_user_folders.ps1` — cópia de pastas de utilizador para diretório de backup;
  - `test_basic_connectivity.ps1` — teste simples de conectividade, DNS e gateway.
- `bash/`
  - `linux_system_inventory.sh` — inventário básico de sistema Linux.
- `python/`
  - `generate_pc_report.py` — relatório simples do sistema em JSON.

## Nota importante
Estes scripts foram pensados para fins de demonstração e aprendizagem. Não substituem ferramentas de gestão empresarial nem administração avançada de infraestruturas.
